First Steps

Activate Plugins

If you want to use all features of DYSANIA (Custom post types,shortcodes,widgets...etc), you should activate the required plugins.After the installation of the theme, you will see a message about required plugins. Click "Begin Installing Plugins", install the plugins and activate them.

Loading Demo Content

If you want to upload demo content (posts,pages,menu...) to your website, download this zip file and extract the xml file. After that use Wordpress import tool to upload this xml file to your website. For more information; Click here

Important; To upload all the demo content properly, you should activate all recommended plugins before uploading the demo xml file.

Note: If you aren't using a fresh Wordpress install, I recommend you reset your database before loading demo content. You can do this easily with this plugin. After doing this don't forget to activate the theme again before loading the xml file...

Selecting Front Page

First of all you should go to Settings->Reading and select in the drop-down box a static page for the front page display. For more information about page templates, click here...

Create a Menu

Dysania supports Wordpress custom menu feature.You can create your menu at Appearance->Menus.For more information click here

Don't forget to check "Main Menu" box.

Add Your Logo

Dysania supports Wordpress custom header feature. You can add your logo at Appearance->Header. Suggested logo height is 97px. For more information click here

Sliders

Zeus Slider

There are two slider option for the homepage in Dysania. One of them is ZeusSlider. When you click "Add new Slide" link, you will see some slider options;

Page Title: If you add 2 image, It will be used as a title of your first slider image.

Slider Image: It is the main image of the slide.

Second Image: If you want to use 2 image in a single slide, you can upload a second image here. If you want to use a single image in a slide, leave this field blank.

Second Title: If you add 2 image, this will be used as a title of your second image. If you add 1 image this will be used as a title of your single image.

Image Animation: It is the css3 animation of the first slide image.

2. Image Animation: It is the css3 animation of the second slide image.

Link: If you want to add a link to the slide, use this field. Correct link format is; http://www.yoursite.com

Embed Video Link: If you want to add an embed video from YouTube,Vimeo etc. use this field. If you add a video link, images,animations and titles will be disabled for this slide.

Page Templates

There are 20 different page template in Dysania-WP. You can select any of them during adding new page/editing page.

Homepage Templates

Homepage-{name} templates are designed for the homepage. The main difference is sliders. Second difference is dark-light color styles.

Last difference is blog-gallery layouts. Normally you can add your latest blog posts to your homepage. In Homepage-{name}-Gallery templates, there is a gallery grid instead of the blog posts. To use these page templates, first of all you should install and activate Dysania Grid Gallery plugin and define the gallery id at Theme Settings.

If you don't want to use blog grid or gallery grid on your homepage, you can disable them at Theme Settings.

Other Templates

Generally there are three different page layout. Half-Width,Full-Dark and Full-Light.

If you select a page template which is using half-width layout, a second content field appears. This field is used for the right side of the page.

Single portfolio page templates are designed for your single portfolio items but you can use these templates for any purpose. You should add featured images to these page templates. I suggest you to use vertical images instead of horizontal images.

Go to Gallery Page

After adding your gallery items, you can go to the related gallery page with clicking "Go to gallery" link or you can click on "Galleries" menu to see all available galleries.

Use the Shortcode

Now you can add this gallery to any page or post with the gallery shortcode which is on the gallery's page. You can change column number of the grid and resolutions of the thumbnails. Default column number is 4 and default thumbnail resolution is 300x300 px.

Hint: You can select thumbnail resolution according to column number. For bigger column numbers, use smaller images for better performance.

For example if you select 2 columns, you should select 600x600 px resolution for better display. If you select 5 columns, you should select smaller image resolution for better page loading performance.

You can also view, edit or delete gallery items directly from edit gallery page.

Filter Menu

You can create filterable galleries with filter menu feature. To use this feature, you should add filters during adding or editing an item.

Adding a filter menu

If you want to add a filter menu to your gallery, you can use "Filters" shortcode. There is a "add a filter menu" icon on your text editor. Click on it. Sample filter menu code will be added your page.

If you want to add shortcodes manually, I recommend you to use "Text mode" of the text editor.

You should add your filters like the following structure;

[filters galleryid="1"]
[filter]Living Room[/filter]
[filter]Bedroom[/filter]
[filter]Bathroom[/filter]
[filter]Office[/filter]
[/filters]

Using multiple filters

To add multiple filters to a page, change the "galleryid" value of the "filters" with the related grid gallery id.

[filters galleryid="1"]
[filter]Living Room[/filter]
[filter]Bedroom[/filter]
[filter]Bathroom[/filter]
[filter]Office[/filter]
[/filters]
[gridgallery id="1"]

        
[filters galleryid="2"]
[filter]Images[/filter]
[filter]Videos[/filter]
[filter]Links[/filter]
[/filters]
[gridgallery id="2"]

Displaying all filters

Add only 1 filter into each [filter] shortcode and be careful when adding words. They must be the same as your filters. You can view,edit or delete filters at Filters page.

How to use widget

Using Dysania grid gallery widget is very easy. Just drag-drop the widget to your sidebar, make your column and resolution choices and click save button. It is totally responsive and fits all widget fields.
